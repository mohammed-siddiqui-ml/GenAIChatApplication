# Infrastructure

This directory contains infrastructure-related files for deployment and orchestration.

## Contents

- **Kubernetes manifests**: k8s configuration files for deployment
- **Terraform scripts**: Infrastructure as Code for cloud resources
- **CI/CD pipelines**: GitHub Actions, GitLab CI, or other CI/CD configurations
- **Monitoring**: Prometheus, Grafana configurations
- **Service mesh**: Istio or other service mesh configurations

## Deployment

Deployment configurations will be added as the project progresses.
